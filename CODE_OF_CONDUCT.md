# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Summary

We are committed to providing a welcoming and inclusive experience for everyone. We expect all participants to be respectful and constructive in all interactions.

## Reporting

If you experience or witness unacceptable behavior, please email brian@potterdigital.com. All reports will be handled confidentially.
